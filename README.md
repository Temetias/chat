# Chat

## Requirements

- node v16.3.0 or greater (Might work on older versions too)

## Get started

- `npm install`
- Fill needed credentials in `src/conf.ts`
- `npm start`
- Open [http://localhost:3000](http://localhost:3000) to view it in the browser. (Should open automagically)
